# file-app
# ファイル整頓アプリ

# アプリ概要
本アプリは、ウェブアプリ内で、PC内に存在するファイルを適切なフォルダに割り当てて整理をしてくれるアプリである。
あらかじめ指定したキーワードが入っている場合には、その該当フォルダに収納される。追加でフォルダを作成したい場合には、追加ボタンから新たなキーワードを入力し、新規フォルダを作成する。また、この機能は1つずつでなく、一括で整頓する機能も持つ。また、フォルダからファイルを消したい場合には、右クリックで選択し削除できる。
