# file-app
# ファイル自動整理アプリケーション

# 現在動く機能
・振り分け元に入っているファイルを指定した特定のキーワードのフォルダに仕分けることができる
・ファイルをドラッグ＆ドロップして選択することができる
・該当しないキーワードを含むファイルは未該当フォルダへ仕分けることができる

#　起動方法
python　app.py

#　動作確認
1.ブラウザでhttps://ruru-00000.github.io/file-app/を開く
2.振り分け元ファイル内から仕分けたいファイルを選択
3.サイト下部に表示される文章通りの仕分けが行われているかを確認
