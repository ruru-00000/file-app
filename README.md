# file-app
# ファイル自動整理アプリケーション

ファイル名に含まれる特定のキーワードを基に、ファイルをフォルダに自動的に仕分け・整理するデスクトップアプリケーションです。

## 主な機能

実行前プレビュー: 実際にファイルを移動する前に、シミュレーション結果（移動先の設定）を確認できます。
元に戻す（Undo）機能: 直前に一括移動したファイルを、ワンクリックで元の配置に復元します。
重複ファイルガード: 移動先に同名のファイルが存在する場合、上書きを防止するため処理を安全に自動停止します。
CSVログ保存: 実行した移動履歴はすべて自動的に `log.csv` へ蓄積されます。
